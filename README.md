# 📊 Netflix EDA Project

## 📌 Overview
This project performs Exploratory Data Analysis (EDA) on a Netflix sample dataset using Python.  
The goal of this project is to understand the dataset, identify patterns, visualize trends, and extract meaningful insights.

---

## 🛠 Tools & Libraries Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Google Colab

---

## 📂 Dataset Information
The dataset contains:
- Movie and TV Show titles
- Genres
- Ratings
- Release Years
- Duration
- Country information

---

## 📊 Exploratory Data Analysis Performed
- Data Loading
- Data Inspection
- Missing Value Check
- Duplicate Check
- Statistical Summary
- Data Visualization
- Correlation Analysis

---

## 📈 Visualizations Included
- Movies vs TV Shows Count
- Genre Distribution
- Release Year Trend
- Duration Analysis
- Correlation Heatmap

---

## 🔍 Key Insights
- Movies are more common than TV Shows in the dataset.
- Some genres dominate the dataset more than others.
- Most content belongs to recent years.
- No strong correlation was found between release year and duration.
- Dataset was mostly clean and structured.

---

## 🚀 Conclusion
This EDA project helped in understanding data analysis workflows including data exploration, visualization, and insight generation using Python.
